# Reproduce everything

This repository is built to be reproduced end-to-end on any machine with Python 3.9+
and no special hardware, network, or data files. Every figure and every number in the
report is regenerated by code.

## TL;DR — one command

```bash
git clone https://github.com/Samarjithbiswas/space-based-ai-datacenter
cd space-based-ai-datacenter
pip install numpy matplotlib pytest
python reproduce_all.py
```

You should see a `PASS` for all seven steps and freshly generated PNGs under
`systems-model/figures/`, `survivability-model/figures/`, and `report/figures/`.

## What each piece does

| Command | What it produces |
|---|---|
| `python systems-model/validate.py` | 23 worked-example checks against the published numbers |
| `pytest -q` (in `systems-model/`) | 83-test suite: worked examples + property tests + integration |
| `python systems-model/run_study.py` | Solves the integrated design point + constellation + Monte-Carlo; writes 6 figures |
| `python survivability-model/validate.py` | 14 worked-example checks |
| `python survivability-model/survivability.py` | The 7 survivability figures |
| `python report/sim_survivability.py` | The 7 report figures |
| `python report/make_equation_images.py` | Equation images (PNG) |

## Use the models in your own code

```python
from orbital_dc.system import DesignPoint, constellation   # systems-model on PYTHONPATH
DesignPoint(n_chips=4, chip="tpu_v6e", alt_km=650).solve()  # one self-consistent satellite
constellation()                                             # aggregate to an 81-sat cluster
```

Every subsystem (`orbit`, `thermal`, `power`, `adcs`, `comms`, `radiation`, `debris`,
`propulsion`, `reliability`, `compute`, `economics`) is an importable module of pure,
documented functions. See `systems-model/docs/METHODS.md` for the equations and
`systems-model/docs/SOURCES.md` for every input with a confidence flag.

## Verifying a single headline number

Want to check, say, the de-orbit delta-v?

```python
from orbital_dc import propulsion
propulsion.deorbit_delta_v(650)   # -> 131.6 m/s  (matches the report and the derivation)
```

Each function's docstring states its governing equation; the full derivation is in the
report's mathematical section. Independent reproduction and corrections are welcome via
issues or pull requests.

## Honest scope

These are first-order *sizing* models with stated uncertainty bands (the ~100x
solar-cycle density swing is the dominant one). They are meant to size problems and rank
risks transparently, not to serve as flight design. For mission work use NASA DAS,
ORDEM/MASTER, SPENVIS/SHIELDOSE-2, and a full thermal/orbit toolchain.
